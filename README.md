# AlphaBull Trading Academy — Next.js Website (Starter)

This is a modern **Next.js (App Router) + TypeScript + Tailwind** website starter.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Where to edit

- Pages live in `app/` (file-based routing)
- Reusable UI components in `components/`
- Static assets (logo, images, brochure) in `public/`

## Contact form

The contact form posts to `/api/contact` (see `app/api/contact/route.ts`).

> Replace the placeholder logic with email (Resend/SMTP) when ready.

## Replace content

All website copy in this starter is **original** and meant to be edited to match your exact services, course names, pricing, legal text, and contact details.
